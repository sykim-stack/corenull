# HajunAI Chrome Extension
BRAINPOOL 관제탑 — 작업 기록 · 문서 저장 · 헬스 모니터링

## 설치 방법

1. Chrome 주소창에 `chrome://extensions` 입력
2. 우측 상단 **개발자 모드** 켜기
3. **압축 해제된 확장 프로그램 로드** 클릭
4. 이 폴더 선택

## 첫 설정

1. 확장프로그램 아이콘 클릭
2. **⚙️ 설정** 탭
3. Supabase URL + Anon Key 입력 후 저장

## Supabase 설정

`documents_table.sql` 을 Supabase SQL Editor에서 먼저 실행하세요.

## 기능

| 탭 | 기능 |
|---|---|
| 📝 작업 기록 | 완료/이슈/결정/메모 → documents 테이블 저장 |
| 📄 문서 저장 | 작업현황/설계문서/보고서/SQL → DB 저장 |
| 📊 헬스 | 6개 프로젝트 health_score 실시간 조회 |
| ⚙️ 설정 | Supabase 연결 설정 |
